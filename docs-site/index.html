What's in the Demo

Four specialist agents, three MCP tool servers, and a full observability pipeline.

Agent

Triage Agent

Front-door router. Greets customers, understands intent, and routes to the correct specialist. Powered by Claude Sonnet. No tools — pure routing logic.

Agent

Customer Service Agent

Handles account inquiries, balance checks, and transaction reviews after verifying customer identity. Powered by GPT-4o Mini with customer and transaction tool access.

Agent

Mortgage Advisor Agent

Provides personalized mortgage rate quotes by looking up customer credit scores, applying rate tables, and checking lending policies. Powered by Claude Sonnet.

Agent

Compliance Agent

Internal-only agent for policy audits and AML/KYC review. Has full access to all customer data, transactions, and bank policies. Generates structured compliance reports.

MCP Server

Customer Tools

Exposes customer lookup, profile retrieval, account balance, and search capabilities over the STREAMABLE_HTTP MCP protocol on port 8081.

MCP Server

Policy & Transaction Tools

Separate MCP servers for bank policy lookups (lending, KYC/AML, rate tables) and transaction history/account analytics — ports 8082 and 8083 respectively.
